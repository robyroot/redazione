# Web Grafica Skills — Settimana 20260622

## 🏆 Top 3 scoperte della settimana

1. **CSS Scroll-Driven Animations nativamente triggered** — Chrome 145 ha introdotto le scroll-triggered animations dichiarative: si attivano al superamento di uno scroll offset, rimpiazzando completamente IntersectionObserver senza una riga di JS. Combinato con `timeline-scope`, è ora possibile orchestrare animazioni tra elementi DOM separati puramente in CSS — un cambio di paradigma.

2. **GSAP 100% free + Motion come standard React 2026** — GSAP (ora gratuito grazie a Webflow) + Motion (ex Framer Motion, motore ibrido 120fps) formano lo stack animation dominante. Stack consigliato: Motion per stato UI React, GSAP per narrative brand, Rive o Lottie per vettoriali designer-authored.

3. **Aurora/Mesh Gradient come estetica SaaS dominante** — Il tool Aurora Gradient Generator di SyntaxSnap genera CSS+HTML Tailwind pronto per Next.js/Astro con blob animati. Zero immagini raster, tutto CSS, performance ottimali. L'estetica "AI premium" con sfondi aurora è diventata il visual language di riferimento per SaaS e AI products.

## Highlight della settimana

Il giugno 2026 segna un momento di maturità per il web grafico: le API CSS native (scroll-driven animations, @property, backdrop-filter) hanno raggiunto supporto universale, riducendo la dipendenza da librerie JS per effetti che prima richiedevano ScrollMagic o GSAP. Al tempo stesso il livello atteso di qualità visiva si è alzato — l'estetica "AI premium" con aurora gradient, glassmorphism 2.0 e tipografia oversized su variable font è diventata lo standard per SaaS e landing page tech. Il tooling AI (Figma Make, Framer AI, Midjourney v7) accelera la produzione di asset e layout al punto che il bottleneck è ora l'integrazione tecnica, non la creatività. Chi padroneggia lo stack CSS nativo + Motion/GSAP + Rive + aurora generator ha tutto il necessario per produrre siti di livello enterprise senza un team di design dedicato.
